# Voorstel Holland Gold — Slimme AI voor het marketingteam

HTML-slidedeck met een voorstel voor een 2-uur durende AI-training voor het marketingteam bij Holland Gold.

## Live bekijken

**https://cschoorl.github.io/voorstel-Holland-Gold/**

> Werkt de link niet (404)? Zet GitHub Pages aan:
> **Settings → Pages → Branch: `main` → Folder: `/ (root)` → Save**
> Na 1–2 minuten is de site bereikbaar.

## Lokaal openen

Open `index.html` in je browser, of start een lokale server:

```bash
cd "Training voorstel Holland Gold"
python3 -m http.server 8080
```

Ga daarna naar http://localhost:8080

## Navigatie

- Pijltjestoetsen of spatiebalk — volgende slide
- Shift + pijltje links — vorige slide
- Knoppen onderaan — vorige / volgende
- Op mobiel — swipen
